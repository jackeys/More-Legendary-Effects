### Hoarder's Effect

**Requires F4SE**

Resistances are added at the following carrying capacity percentages:
* 80% - +4
* 85% - +8
* 90% - +12
* 95% - +16
* 100% or more - +20

### Traveler's Effect

**Requires F4SE**

Movement speed increases at the following carrying capacity percentages:
* 70% - +2% movement speed
* 65% - +4% movement speed
* 60% - +6% movement speed
* 55% - +8% movement speed
* 50% or lower - +10% movement speed
